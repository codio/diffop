# DiffOp

[![Build Status](https://travis-ci.org/codio/diffop.svg?branch=master)](https://travis-ci.org/codio/diffop)

> Transform json diffs into ShareJS operations


## Usage

```js
var diffop = require('diffop');

var ops = diffop(old, new);
```
